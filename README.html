Overview

Java helper library for TUNE Reporting API services.

The utility focus of this SDK is upon the Advertiser Reporting endpoints.

Even though the the breadth of the Management API goes beyond just reports, it is these endpoints that our customers primarily access.

The second goal of the SDKs is to assure that our customersâ€™ developers are using best practices in gathering reports in the most optimal way.

Installation

Maven:

TUNE Reporting API SDK for Java is available using Maven. At present the jar is available from a public Maven repository.

Use the following dependency in your project:

            
               com.tune.reporting
               tune-reporting
               0.9.9
               compile

Github Compile:

If you want to compile it yourself, here's how:

            $ git clone git@github.com:MobileAppTracking/tune-reporting-java
            $ cd tune-reporting
            $ mvn install       # Requires maven, download from http://maven.apache.org/download.html

Via ZIP file:

Click here to download the source code (.zip) for `tune-reporting`.

Code Samples

SDK Examples

Run the following script to view execution of all examples:

            $ make api_key=[API_KEY] ant-examples

SDK Unittests

Run the following script to view execution of all unittests: 
            $ make ant-test api_key=[API_KEY]
        
            $ make mvn-test api_key=[API_KEY]

SDK Documentation

The following will generate Javadoc from java codebase: 
            $ make docs-javadoc
The following will generate Doxygen from java codebase: 
            $ make docs-doxygen
