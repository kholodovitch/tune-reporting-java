Overview

Tune API client for PHP developers. The utility focus of the SDKs is upon the Advertiser Reporting endpoints. Even though the the breadth of the Management API goes beyond just reports, it is these endpoints that our customers primarily access. The second goal of the SDKs is to assure that our customersâ€™ developers are using best practices in gathering reports in the most optimal way.

Code Samples

SDK Examples

Run the following script to view execution of all examples:

            $ make API_KEY=[API_KEY] examples

SDK Unittests

Run the following script to view execution of all unittests: 
            $ make API_KEY=[API_KEY] tests

SDK Documentation

The following will generate Javadoc from java codebase: 
            $ make docs-javadoc
The following will generate Doxygen from java codebase: 
            $ make docs-doxygen
